# Kazoo 5 Development

## What this is...

* A development and integration repo for everything Kazoo 5-related
* A proving ground for new features, architectural changes, and experimental work
* A staging area for incomplete implementations and active development

## What this is not...

* Stable
* Intended for production deployment

In a word, this repo lives in a time paradox. History may be rewritten. Commits may vanish.

## Goals

* Enable Kazoo 5 to stand on its own without the need for closed-source modules, proprietary features, or lock-in.
* Allow the community to inspect the source code and deploy Kazoo on whatever OS they like without limitation.
* Grow participation from the community to set the direction of the project and development of new features.

Over time, the work done in this repo will be cherry-picked into the appropriate repos, packaged, and distributed.

## Contributing

All issues, pull requests, ideas, and thoughts are welcome.
